Python Concepts Review for Kahoot Quiz

Here's a review of some important Python concepts that will be covered in tomorrow's Kahoot quiz. These concepts are related to lists and dictionaries.

1. The extend() method for lists

This method adds all the elements of an iterable (like another list) to the end of the current list.

fruits = ['apple', 'banana']
more_fruits = ['cherry', 'date']
fruits.extend(more_fruits)
print(fruits)  # Output: ['apple', 'banana', 'cherry', 'date']

2. The sorted() function

This function returns a new sorted list from a given iterable. It can sort in ascending or descending order.

numbers = [3, 1, 4, 1, 5, 9, 2]
sorted_ascending = sorted(numbers)
sorted_descending = sorted(numbers, reverse=True)
print(sorted_ascending)   # Output: [1, 1, 2, 3, 4, 5, 9]
print(sorted_descending)  # Output: [9, 5, 4, 3, 2, 1, 1]

3. The 'in' expression for dictionaries

This is used to check if a key exists in a dictionary.

student_grades = {'Alice': 85, 'Bob': 92, 'Charlie': 78}
if 'Bob' in student_grades:
    print("Bob's grade:", student_grades['Bob'])
else:
    print("Bob's grade not found")
